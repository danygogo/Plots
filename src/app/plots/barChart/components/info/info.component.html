Title

The title will appear at the top of your chart. It is optional.

Dataset name

It is the name that you will add to your serie, for example, if you will draw the results for the behavior of women and men in some activity, you could write "women" or "men" in the data set name.

This information is requiered.

Quantity of points

This value is the quantity of measures that you have, for example, if you what to show the women's results in three diferent days then write 3.

This information is requiered and the minimun is 1.

Bar's color

Pick the color and your bar will be this color.

Text color for labels

This is the color that will appear on the name of your dataset, it will be at the top of your chart.

Labels' color on x axis

This color will be used on the numbers on x-axis.

Labels' color on y axis

This color will be used on the numbers on y-axis.

Vertical lines' color

This color will be used in the vertical lines of your chart. Pick white if you don't want to see them.

Horizontal lines' color

This color will be used in the horizontal lines of your chart. Pick white if you don't want to see them.

Quantitative information

In this table you have to add the data that will fill the chart. Write down the values that you want in x-axis (vertically), for example, if you are working with the women's results in 3 different days then write down 1, 2 and 3.

Fill in the same way (vertically) the values for the y-axis, in our example, you should write down the women's results.

All the fields are required.
