Title

The title will appear at the top of your chart. It is optional.

Number of observations

It is the number of elements or observations that you want to add to your chart. For example, if you want to represent in the chart the number of women and men in your chart, then write 2

this value is requiered and the minimun is 1.

Table

In the table below you have to add your data. For every type of data you have to add the values vertically, for example if you are showing the number of the men and women in your class, then write vertically "men" and "women". Do the same for the other data.

The numbers must be greater than or equal to zero.

Pick different colors for each data.

All the fields are required.
